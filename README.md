# FinTrack

**Live Site:** [https://fintrack-ph.netlify.app](https://fintrack-ph.netlify.app)

FinTrack is a modern web application for tracking and managing personal finances. It helps users organize expenses, monitor income, and analyze sr visual insighpending patterns through cleats.

## Features

- Interactive charts and graphs powered by **Recharts** for data visualization.
- Secure user authentication and cloud storage with **Firebase**.
- Real-time notifications and alerts using **React Toastify** and **SweetAlert2**.
- Fast and optimized performance with **Vite** and **React 19**.
- Clean and responsive UI styled with **Tailwind CSS** and **DaisyUI**.

## Tech Stack

- **Frontend:** React 19, Tailwind CSS, DaisyUI
- **Backend & Storage:** Firebase
- **Visualization:** Recharts
- **Build Tool:** Vite
- **Utilities:** Axios, SweetAlert2, React Toastify
